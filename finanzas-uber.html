Configuración de Firebase

1. Ve a console.firebase.google.com y crea un proyecto gratis.
2. Activa Realtime Database (modo test).
3. En Configuración del proyecto → copia los datos y pégalos aquí.

▶ ¿Cómo configurar las reglas de Firebase?
En Firebase Console → Realtime Database → Reglas, pega esto y publica:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
Para uso personal. Nadie más sabrá tu Database URL.
¿Cómo funciona?
  • Cada vez que guardas un dato se sincroniza automáticamente a Firebase.
  • Al abrir la app en cualquier dispositivo carga los datos desde la nube.
  • Los datos locales (localStorage) se mantienen como respaldo offline.
  • Puedes seguir usando Exportar/Importar JSON como backup adicional.
